# Antlr parser in Angular
This is simple web based parser generated by antlr and antlr4ts in angular.

An example of the grammar:
```
<chess_game>
    <player>
        <name> ali </name>
        <code> 2 </code>
        <color> black </color>
    </player>
    <player>
        <name> alim </name>
        <code> 1 </code>
        <color> black </color>
    </player>
    <result>
        <winner> 1 </winner>
        <time> 200000 </time>
        <moves> 10 </moves>
    </result>
</chess_game>
```
